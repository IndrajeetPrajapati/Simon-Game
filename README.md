# Simon Game

A fun and interactive memory game built using **HTML, CSS, JavaScript. The game challenges players to remember and repeat an increasingly complex sequence of colors and sounds.

## 🎮 Features

* Random color sequence generation
* Sound effects for each button
* Level progression system
* Game over animation and restart option
* Keyboard start functionality
* Responsive and simple UI

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📌 How to Play

1. Press any key to start the game.
2. Watch the highlighted color pattern.
3. Repeat the same sequence by clicking buttons.
4. The sequence grows each level.
5. One wrong click ends the game.

## 🚀 Purpose

This project was created to practice JavaScript logic, DOM manipulation, event handling, and game development fundamentals.
